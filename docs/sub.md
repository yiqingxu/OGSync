# OGSync sub commands

## OGSync home

Open the OGSync Home Page and exit

```
OGSync.py home
```

optional arguments:
| argument | description |
| --- | --- |
| `-h`, `--help` | show this help message and exit |
| `--debug` | display debug info, for developers only |

source: [oglib/og_home_impl.py](../oglib/og_home_impl.py)
